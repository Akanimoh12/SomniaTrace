# SomniaTrace

Real-time blockchain explorer for the **Somnia Network** with interactive bubble visualization. Watch transactions flow across the network as dynamic, physics-driven bubbles — search any wallet, switch networks, zoom, and expand for a clearer view.

![SomniaTrace Preview](./public/preview.png)

## Features

- **Live Transaction Bubbles** — Real-time D3.js force simulation visualizing on-chain activity
- **Wallet Search** — Look up any address to see its transaction history as bubbles
- **Network Switching** — Toggle between Somnia Devnet and Testnet at runtime
- **Zoom & Fullscreen** — Scroll to zoom, expand the canvas for a detailed view
- **Transaction Panel** — Side panel with live transaction feed and details
- **Token Tracking** — Monitors PFT (ERC-20) transfers on Somnia Testnet

## Tech Stack

Next.js 14 · TypeScript · Tailwind CSS · D3.js · viem

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
